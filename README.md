# Air Cargo Analysis using SQL

## Project Overview
This project analyzes airline cargo operations using MySQL to generate insights on passenger trends, revenue performance, route efficiency, and ticket class analysis.

## Tools & Technologies
- MySQL
- Window Functions
- Stored Procedures
- Stored Functions
- Indexing & Performance Optimization
- Aggregations & Grouping
- Views & Cursors

## Key Analysis Performed
- Revenue analysis by ticket class
- Passenger analysis across route ranges
- Maximum ticket price using window functions
- Route categorization by travel distance
- Complimentary service identification logic
- Revenue threshold analysis
- Performance improvement using indexing and EXPLAIN
- Rollup for hierarchical revenue aggregation

## Advanced SQL Concepts Used
- Joins
- Group By & Having
- Subqueries
- Window Functions
- Stored Procedures
- Stored Functions
- Indexing
- Execution Plan Analysis
- Rollup
- Constraints

## Outcome
This project demonstrates strong SQL skills along with business-oriented data analysis thinking, suitable for Data Analyst and Business Analyst roles.
